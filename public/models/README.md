# Veyn Room Assets

Drop production room and avatar assets here:

- `agent-ops-room.glb`
- `avatars/manager.glb`
- `avatars/researcher.glb`
- `avatars/coder.glb`
- `avatars/reviewer.glb`
- `avatars/deployment.glb`
- `avatars/support.glb`

The room renderer falls back to procedural geometry when these files are not present.
